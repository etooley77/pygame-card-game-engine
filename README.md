# PyGame Card Engine

A side-project game engine built onto PyGame meant to support card game development. The engine (so far) is tailored towards a programmer-oriented development style.

---

## Features

 - [x] Hoverable and Draggable CardView with proper z-order
 - [x] CardView snapping and SnapZone

---

## Planned Features

 - [ ] Hand / Inventory System
 - [ ] Art for the full default deck including Jokers
 - [ ] Custom deck implementation with config and manager system
